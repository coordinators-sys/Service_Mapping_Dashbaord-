Matched by Site ID — the record is linked to an official site on the CCCM master list by its permanent Site ID (or exact name). Trust these; they feed the coverage figures directly.

Needs review — linked only by an approximate site name. Probably correct, but a person should confirm it. Still counted, and flagged for cleanup.

Unmatched — could not be linked to any master-list site. Usually the form carried no Site ID (or a code from a different tool), or the site is not yet registered in the master list. These are shown for transparency and excluded from master-list coverage counts.

"Unknown" sectors — a sector left blank on the form. It is never counted as "not covered"; it is excluded from the coverage-rate denominator so a blank can't understate coverage.
